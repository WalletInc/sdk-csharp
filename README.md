# WalletInc .NET SDK

Auto-generated .NET client for the Wallet Inc platform, published to NuGet as **WalletInc**.

The source in this repository is generated by the wallet-ts-api CI (openapi-generator, `-g csharp`)
on each deploy and pushed to the matching branch (`dev` / `staging` / `master`). Do not hand-edit
generated files. The NuGet publish workflow lives in `.github/workflows/ci.yml` and runs on `master`.

Docs: https://wallet.dev
